Latest changes in version 9.2:

Improve use of google-translate feature on UniqueBible.app:
* changed "myLanuage" to "userLanguage" in config.py
* added a gui dialog for users to define their target language for Google Translate.  The dialog is accessible via application menu.

Remarks: To use google translate on UniqueBible.app, you have to install a python package called 'googletrans'.  To install, run "pip3 install googletrans" in command prompt.
